13 // Proof-of-concept implemenation of the Quantum Optimization of Analytic conTrols (GOAT) algorithm.
14 // The GOAT algorithm required:
15 // (1) Analytical expression of the differentiation of the Hamiltonian w.r.t. the control params.
16 // (2) A generic integrator (e.g. Runge-Kutta) to evolve the U matrix and its differentiation w.r.t. control params.
17 // This needs the result from (1) to construct the RHS of the differential equation.
18 // (3) A gradient-based param search mechanism (e.g. BFGS)
19 // The gradient of the cost/target function w.r.t. control params is calculated by the result of (2)
20 // All of these ingredients are abstracted away, i.e. provided by user.
21 // TODO: gradually put this into a more structured framework (perhaps as services)
22
